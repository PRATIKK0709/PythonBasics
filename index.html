Welcome to the complete Python documentation. This guide will take you from absolute beginner to professional Python developer. Everything you need in one place.

What is Python?

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum in 1991, it emphasizes code readability and allows developers to express concepts in fewer lines of code.

Why Learn Python?

  • Easy to Learn: Simple, readable syntax similar to English
  • Versatile: Web development, data science, automation, AI/ML, scripting
  • Huge Ecosystem: Over 400,000 packages available via PyPI
  • Industry Demand: Used by Google, Netflix, Instagram, NASA, Spotify
  • Great Community: Massive support network and resources

What You'll Learn

This documentation covers:

  • Python fundamentals and syntax
  • Data structures and algorithms
  • Object-oriented programming
  • File operations and exception handling
  • Web development with Flask and FastAPI
  • Discord bot development with discord.py
  • Data science with NumPy and Pandas
  • Async programming and concurrency
  • Best practices and professional patterns

Python Syntax #

Indentation

Python uses indentation to define code blocks. This is not optional.

# Correct
if 5 > 3:
    print("Five is greater")
    print("This is also in the block")

# Wrong - IndentationError
if 5 > 3:
print("This will fail")

Standard: Use 4 spaces for indentation (PEP 8). Most editors can convert tabs to spaces automatically.

Comments

# Single line comment

Multi-line comment
or docstring
"""

x = 5  # Inline comment

Line Continuation

# Using backslash
total = 1 + 2 + 3 + \
        4 + 5

# Implicit continuation (preferred)
total = (1 + 2 + 3 +
         4 + 5)

Multiple Statements

# Multiple statements on one line (avoid)
x = 5; y = 10; print(x + y)

# Better: one per line
x = 5
y = 10
print(x + y)

  • Case-sensitive (age and Age are different)

# snake_case for variables and functions

    pass

# PascalCase for classes
class UserAccount:
    pass

Operators #

Arithmetic Operators

a = 10
b = 3

print(a + b)   # 13 - Addition
print(a - b)   # 7  - Subtraction
print(a * b)   # 30 - Multiplication
print(a / b)   # 3.333... - Division
print(a // b)  # 3  - Floor Division
print(a % b)   # 1  - Modulus
print(a ** b)  # 1000 - Exponentiation

Comparison Operators

x = 5
y = 10

print(x == y)  # False - Equal
print(x != y)  # True  - Not equal
print(x > y)   # False - Greater than
print(x < y)   # True  - Less than
print(x >= y)  # False - Greater or equal
print(x <= y)  # True  - Less or equal

Logical Operators

a = True
b = False

print(a and b)  # False
print(a or b)   # True
print(not a)     # False

Assignment Operators

x = 10
x += 5   # x = x + 5
x -= 3   # x = x - 3
x *= 2   # x = x * 2
x /= 4   # x = x / 4
x //= 2  # x = x // 2
x %= 3   # x = x % 3
x **= 2  # x = x ** 2

Membership Operators

fruits = ["apple", "banana", "cherry"]

print("apple" in fruits)      # True
print("grape" not in fruits)  # True

Identity Operators

x = [1, 2, 3]
y = [1, 2, 3]
z = x

print(x is z)      # True  - Same object
print(x is y)      # False - Different objects
print(x == y)       # True  - Same values
print(x is not y)  # True

Bitwise Operators

a = 60  # 0011 1100
b = 13  # 0000 1101

print(a & b)   # 12 - AND
print(a | b)   # 61 - OR
print(a ^ b)   # 49 - XOR
print(~a)      # -61 - NOT
print(a << 2)  # 240 - Left shift
print(a >> 2)  # 15 - Right shift

Operator Precedence

From highest to lowest:

  1. Parentheses: ()
  2. Exponentiation: **
  3. Unary: +x, -x, ~x
  4. Multiplication/Division: *, /, //, %
  5. Addition/Subtraction: +, -
  6. Bitwise shifts: <<, >>
  7. Bitwise AND: &
  8. Bitwise XOR: ^
  9. Bitwise OR: |
  10. Comparison: ==, !=, <, >, <=, >=, is, in
  11. Logical NOT: not
  12. Logical AND: and
  13. Logical OR: or
result = 2 + 3 * 4      # 14 (not 20)
result = (2 + 3) * 4    # 20
result = 2 ** 3 ** 2     # 512 (right associative)

Strings #

Strings are sequences of characters used for text.

Creating Strings

# Single or double quotes
name = 'Alice'
message = "Hello, World!"

# Multi-line strings
text = """This is
a multi-line
string"""

# Escape characters
quote = "He said \"Hello\""
path = "C:\\Users\\Documents"  # Backslash
newline = "Line 1\nLine 2"

# Raw strings (no escape processing)
regex = r"\d+\.\d+"

String Operations

# Concatenation
first = "Hello"
last = "World"
full = first + " " + last  # "Hello World"

# Repetition
laugh = "Ha" * 3  # "HaHaHa"

# Length
print(len("Python"))  # 6

String Indexing and Slicing

text = "Python"

# Indexing (0-based)
print(text[0])   # 'P'
print(text[-1])  # 'n' (last character)

# Slicing [start:end:step]
print(text[0:3])   # 'Pyt'
print(text[2:])     # 'thon'
print(text[:4])     # 'Pyth'
print(text[::2])    # 'Pto' (every 2nd char)
print(text[::-1])   # 'nohtyP' (reverse)

String Methods

text = "  Hello, World!  "

# Case conversion
print(text.upper())       # "  HELLO, WORLD!  "
print(text.lower())       # "  hello, world!  "
print(text.title())       # "  Hello, World!  "
print(text.capitalize())  # "  hello, world!  "
print(text.swapcase())    # "  hELLO, wORLD!  "

# Whitespace handling
print(text.strip())   # "Hello, World!"
print(text.lstrip())  # "Hello, World!  "
print(text.rstrip())  # "  Hello, World!"

# Search and replace
print(text.find("World"))     # 9
print(text.replace("World", "Python"))  # "  Hello, Python!  "
print(text.count("l"))       # 3

# Split and join
words = text.strip().split(", ")  # ['Hello', 'World!']
joined = "-".join(words)          # "Hello-World!"

# Checking
print("hello".isalpha())   # True
print("123".isdigit())     # True
print("abc123".isalnum())  # True
print("   ".isspace())     # True
print("Hello".startswith("He"))  # True
print("Hello".endswith("lo"))    # True

String Formatting

name = "Alice"
age = 25
price = 19.99

# f-strings (Python 3.6+, preferred)
msg = f"My name is {name} and I'm {age} years old"
print(msg)

# Expressions in f-strings
print(f"Next year I'll be {age + 1}")
print(f"Price with tax: ${price * 1.08:.2f}")

# Format method
msg = "My name is {} and I'm {} years old".format(name, age)
msg = "My name is {n} and I'm {a} years old".format(n=name, a=age)

# Old style (avoid)
msg = "My name is %s and I'm %d years old" % (name, age)

Format Specifiers

num = 42
pi = 3.14159

print(f"{num:5}")      # "   42" (width 5)
print(f"{num:05}")     # "00042" (zero-padded)
print(f"{pi:.2f}")     # "3.14" (2 decimals)
print(f"{num:b}")      # "101010" (binary)
print(f"{num:x}")      # "2a" (hexadecimal)
print(f"{1000:,}")     # "1,000" (thousands sep)

Numbers #

Python supports three types of numbers:

  • int: Whole numbers (e.g., 10, -5, 1000)
  • float: Decimal numbers (e.g., 3.14, -0.01)
  • complex: Numbers with a real and imaginary part (e.g., 2+3j)

Arithmetic Operations

x = 10
y = 3

print(x + y)  # 13 (Addition)
print(x - y)  # 7 (Subtraction)
print(x * y)  # 30 (Multiplication)
print(x / y)  # 3.333... (Division)
print(x // y) # 3 (Floor Division)
print(x % y)  # 1 (Modulus/Remainder)
print(x ** y) # 1000 (Exponentiation)

Note: Python integers have unlimited precision, meaning they can be as large as your computer's memory allows.

Booleans #

Booleans represent one of two values: True or False. They are commonly used in conditional statements.

Comparison Operations

print(10 > 9)   # True
print(10 == 9)  # False
print(10 < 9)   # False

Truthy and Falsy Values

In Python, the following are considered False when evaluated in a boolean context:

  • None
  • False
  • Zero of any numeric type (0, 0.0, 0j)
  • Empty sequences/collections ('', (), [], {}, set())

Almost everything else is considered True.

Type Conversion #

You can convert between different data types using constructor functions.

Implicit Conversion

Python automatically converts lower precision types to higher precision types to avoid data loss. 

x = 1    # int
y = 2.5  # float
z = x + y
print(z)      # 3.5
print(type(z)) # <class 'float'>

Explicit Conversion (Casting)

# Int to Float
a = float(1)    # 1.0

# Float to Int
b = int(2.8)    # 2 (truncates decimal part)

# Number to String
c = str(10)     # "10"

# String to Number
d = int("5")    # 5
e = float("3.5") # 3.5

Lists #

Lists are ordered, mutable collections of items. They allow duplicate elements.

Creating Lists

fruits = ["apple", "banana", "cherry"]
numbers = [1, 5, 7, 9, 3]
mixed = [1, "hello", True, 3.14]

Accessing Items

print(fruits[0])      # "apple"
print(fruits[-1])     # "cherry" (last item)
print(fruits[1:3])    # ["banana", "cherry"] (slicing)

Modifying Lists

fruits[1] = "blueberry"    # Change item
fruits.append("orange")    # Add to end
fruits.insert(1, "kiwi")   # Add at index
fruits.remove("apple")     # Remove item
last = fruits.pop()        # Remove and return last item

List Comprehension

A concise way to create lists based on existing lists.

# Create list of squares
squares = [x**2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]

Tuples #

Tuples are ordered collections like lists, but they are immutable (cannot be changed after creation).

Creating Tuples

point = (10, 20)
colors = ("red", "green", "blue")

Unpacking

Use Case: Use tuples for data that shouldn't change, like geographic coordinates or RGB color values.

Sets are unordered collections of unique items.

Creating and Modifying

For Loops #

Used for iterating over a sequence (list, tuple, dictionary, set, or string).

fruits = ["apple", "banana", "cherry"]
for x in fruits:
    print(x)

Range Function

for i in range(5):
    print(i)  # 0, 1, 2, 3, 4

for i in range(2, 10, 2):
    print(i)  # 2, 4, 6, 8 (step 2)

Enumerate

for index, value in enumerate(fruits):
    print(f"{index}: {value}")

While Loops #

Executes a set of statements as long as a condition is true.

i = 1
while i < 6:
    print(i)
    i += 1

Break & Continue #

Break

Stops the loop completely.

for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0 to 4

Continue

Skips the current iteration and continues with the next.

for i in range(5):
    if i == 2:
        continue
    print(i)  # Prints 0, 1, 3, 4

Parameters & Arguments #

Default Arguments

def power(base, exponent=2):
    return base ** exponent

Keyword Arguments (**kwargs)

def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

print_info(name="John", age=30)

Return Values #

Single Value

Lambda Functions #

Small anonymous functions defined with the lambda keyword.

# Syntax: lambda arguments : expression
double = lambda x: x * 2

print(double(5))  # 10

Use with map() and filter()

nums = [1, 2, 3, 4, 5]

# [1, 4, 9, 16, 25]

evens = list(filter(lambda x: x % 2 == 0, nums))
# [2, 4]

Decorators #

Decorators allow you to modify the behavior of a function or class.

def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()

Generators #

Generators are functions that return an iterator using yield. They are memory efficient.

def count_up_to(n):
    count = 1
    while count <= n:
        yield count
        count += 1

counter = count_up_to(5)
for num in counter:
    print(num)

Generator Expression

squares = (x*x for x in range(10))
# Similar to list comprehension but returns generator object

Classes & Objects #

Classes are blueprints for creating objects.

class Dog:

        self.name = name

Inheritance #

Allows a class to inherit attributes and methods from another class.

class Animal:
    def speak(self):
        print("Generic animal sound")

class Cat(Animal):
    def speak(self):
        print("Meow")

cat = Cat()
cat.speak()  # Meow

Polymorphism #

The ability to use a common interface for multiple forms (data types).

animals = [Dog("Buddy", "Pug"), Cat()]

for animal in animals:
    print(animal.speak())  # Different output for each

Encapsulation #

Restricting access to methods and variables. In Python, use underscores.

class Account:
    def __init__(self):
        self._protected = 10  # Suggests internal use
        self.__private = 20   # Mangled name (harder to access)

    def get_private(self):
        return self.__private

Magic Methods #

Special methods with double underscores (dunder methods).

class Book:
    def __init__(self, title, pages):
        self.title = title
        self.pages = pages

    def __str__(self):
        return f"{self.title} ({self.pages} pages)"

    def __len__(self):
        return self.pages

b = Book("Python 101", 200)
print(b)        # Python 101 (200 pages)
print(len(b))   # 200

Exception Handling #

Handle runtime errors gracefully using try-except blocks.

try:
    x = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
except Exception as e:
    print(f"An error occurred: {e}")
finally:
    print("This always runs")

Cogs & Extensions #

Cogs are a way to organize commands and listeners into different files/modules.

Creating a Cog

# cogs/moderation.py
from discord.ext import commands
import discord

class Moderation(commands.Cog):
    """Moderation commands"""
    
    def __init__(self, bot):
        self.bot = bot
    
    @commands.command()
    @commands.has_permissions(kick_members=True)
    async def kick(self, ctx, member: discord.Member, *, reason=None):
        await member.kick(reason=reason)
        await ctx.send(f'{member} has been kicked. Reason: {reason}')
    
    @commands.command()
    @commands.has_permissions(ban_members=True)
    async def ban(self, ctx, member: discord.Member, *, reason=None):
        await member.ban(reason=reason)
        await ctx.send(f'{member} has been banned. Reason: {reason}')
    
    @commands.Cog.listener()
    async def on_member_join(self, member):
        channel = member.guild.system_channel
        if channel:
            await channel.send(f'Welcome {member.mention}!')

# Required function for loading
async def setup(bot):
    await bot.add_cog(Moderation(bot))

async def get_coins(user_id):
    async with aiosqlite.connect('bot.db') as db:

            result = await cursor.fetchone()
            return result[0] if result else 0

class Paginator(discord.ui.View):

@bot.event
async def on_raw_reaction_add(payload):
    if payload.member == bot.user:
        return
    
    guild = bot.get_guild(payload.guild_id)
    
    role_map = {
        "": discord.utils.get(guild.roles, name="Gamer"),
        "": discord.utils.get(guild.roles, name="Artist"),
        "": discord.utils.get(guild.roles, name="Developer")
    }
    
    role = role_map.get(str(payload.emoji))
    if role:
        await payload.member.add_roles(role)

@bot.event
async def on_raw_reaction_remove(payload):
    guild = bot.get_guild(payload.guild_id)
    member = guild.get_member(payload.user_id)
    
    role_map = {
        "": discord.utils.get(guild.roles, name="Gamer"),
        "": discord.utils.get(guild.roles, name="Artist"),
        "": discord.utils.get(guild.roles, name="Developer")
    }
    
    role = role_map.get(str(payload.emoji))
    if role and member:
        await member.remove_roles(role)

    if len(message.content) > 500:
        await message.delete()
        await message.channel.send(f'{message.author.mention} No spam!', delete_after=3)
        return
    
    # Bad words filter
    bad_words = ['badword1', 'badword2']
    if any(word in message.content.lower() for word in bad_words):
        await message.delete()
        await message.author.timeout(discord.utils.utcnow() + timedelta(minutes=5))
        return
    
    await bot.process_commands(message)

Common Errors & Solutions #

Python Errors

# IndentationError
# Problem: Incorrect indentation
if True:
print("Wrong")  # Error!

# Solution: Use 4 spaces
if True:
    print("Correct")

# NameError
# Problem: Variable doesn't exist
print(x)  # Error if x not defined

# Solution: Define variable first
x = 10
print(x)

# TypeError
# Problem: Wrong type operation
result = "5" + 5  # Error!

# Solution: Convert types
result = int("5") + 5  # 10

# IndexError
# Problem: Index out of range
lst = [1, 2, 3]
print(lst[10])  # Error!

# Solution: Check length or use try/except
if len(lst) > 10:
    print(lst[10])

# KeyError
# Problem: Dictionary key doesn't exist
d = {"name": "Alice"}
print(d["age"])  # Error!

# Solution: Use .get() or check first
print(d.get("age", "Not found"))  # "Not found"

Discord.py Errors

# Missing Intents
# Problem: Bot can't see message content
bot = commands.Bot(command_prefix='!', intents=discord.Intents.default())

# Solution: Enable message content intent
intents = discord.Intents.default()
intents.message_content = True
bot = commands.Bot(command_prefix='!', intents=intents)

# Problem: Bot can't perform action
# Solution: Check bot permissions in server settings

try:
    await member.kick()
