## Python Syntax

Python has a clean and straightforward syntax. Indentation is crucial for defining blocks of code. Here's a basic if-else statement:

```
Copy code
x = 10

if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")
```
